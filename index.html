Data Type & Format Specifier

A computer processor alows different types of data to calculate something. In C Programme there also some specific data types. Some of them are char, int, float and a lot more.

char

It is the first four letters of character.They can only contain a single letter ('a', 'D', '!', '5').They hold 1byte storage in computer memory.

int

Having first three letters of integer, it is named int. They hold 4bytes storage of computer memory. That means you can store 232 or 4294967296 variety of integer numbers

float

It is used to store floating point number. They also hold 4 bytes storage of computer memory.

Input Output Statement

As a programmer,you have to always work with input and output.In this section we will learn how to take input from user and display some output.

To display anything as output you have to use printf() function.This is a function which is defined in your header function.In this function everything will be displayed that is written within double quotation(" ").

printf("Hello world!")

This programme will so the following output.

Hello world!

In same way you can take input from user.So that you have to use scanf() function.It is same like printf() function.Within the double quotation(" ") you have to write the format specifier as you learnt before. And out side the quotation you have to define each format specifier with respective variable name with comma(,). 

 int number_1=3;
 int number_2=5;
 scanf("%d\n%d",number_1,
 number_2);

The code will show the following output.

_

It means your programme is ready to take input now.

First Programme

Let's start programing! In this section we will get introduced to a formal structure of C Programme and also run a programme.

#include stdio.h(*wrap with this <>)
int main(){

printf("This is my first 
programme.");

return 0;
}

This is the output.


This is my first programme.

In the following code we can see several unknown terms.Now we'll learn them one by one.

In the second line you can see main() it is a main function. Every C programme starts running with this function.Curly brackets define that the main function begins at 2nd line and ends at 8th line.

After that, the printf() function is also a function that you learnt before.But how this function will work it wasn’t defined anywhere else. The stdio.h is such a header file which defines how the functions in this programme will work.

In the first line #include stdio.h(*wrap with this <>) defines that now you can use the header file in your programme.

If...else Statement

Now we'll learn conditional statement.In this case we'll set our programme to work on different condition.In C we'll use If...else statement.

It has the following syntax. 

If(conditional expression)
{
...Statement 1...
}
else{
...Statement 2...
}

HereStatement 1 being applied when condition is true neither Statement 2 gets applied.
